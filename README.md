# El-sesgo-en-la-Inteligencia-Artificial
Documentación y ejemplos prácticos sobre la importancia de la ética profesional en el desarrollo de software y la detección de sesgos algorítmicos.
Explora cómo los algoritmos, aunque parezcan neutrales, pueden heredar prejuicios humanos y discriminar grupos sociales. La formación socio-crítica busca cuestionar la responsabilidad del programador en la creación de software justo, transparente y orientado al bienestar común en lugar del control social.
